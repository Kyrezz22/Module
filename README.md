# Performance Monitor App

## Introduction
The Performance Monitor App is designed to track and analyze the performance of various system resources. Users can monitor CPU usage, memory consumption, disk activity, and network speeds in real-time.

## Features
- **Real-Time Monitoring**: Get live statistics about your system's performance.
- **Historical Data**: View and analyze performance data over time.
- **Alerts**: Set up custom alerts for unusual system behavior.
- **User-Friendly Interface**: An intuitive interface that is easy to navigate.

## Technologies Used
- **Programming Language**: Java
- **Framework**: Spring Boot for the backend, React for the frontend
- **Database**: MongoDB for data storage

## Installation
1. Clone the repository: `git clone https://github.com/Kyrezz22/Module.git`
2. Navigate to the project directory: `cd Module`
3. Build the project: `mvn install`
4. Run the application: `java -jar target/performance-monitor-app.jar`

## Usage
Upon launching the application, the dashboard will display real-time statistics. Users can access historical data from the menu and configure alerts through the settings section.
